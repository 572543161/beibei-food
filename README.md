# 贝贝的小食铺 PWA 单机版

这是一个独立于微信小程序的新版本。它不使用微信云开发，不需要登录，不上架 App Store，菜品、图片、菜单和计划都会保存在当前设备的浏览器本地数据库里。

## 在电脑上预览

双击 `start-local.bat`，然后打开浏览器访问对应地址。

常用地址会是：

```text
http://localhost:5177
```

## 在 iPhone 上使用

最稳的长期方式是把整个 `pwa` 目录发布到免费的 HTTPS 静态托管，比如 GitHub Pages。托管服务只放应用代码，不保存你的菜品数据。

发布后：

1. 用 iPhone 的 Safari 打开 HTTPS 地址。
2. 点 Safari 分享按钮，选择“添加到主屏幕”。
3. 以后从桌面图标进入使用。

本地 `http://电脑IP:5177` 适合临时预览，但 iPhone 的离线缓存通常需要 HTTPS 才稳定。

## 数据备份

进入应用底部“备份”页：

- 点“导出备份文件”保存 JSON。
- 换设备或误删后，用“导入备份文件”恢复。

浏览器清理网站数据、删除这个主屏幕网页应用、或 iOS 大扫除缓存，都可能影响本地数据，所以建议定期导出备份。
